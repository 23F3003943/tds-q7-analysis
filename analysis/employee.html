Q7 - Employee Department Analysis

This HTML file contains the Python code, the printed result, and a histogram.

Printed Output

Sales Department Count: 13

Python Code

import pandas as pd
import matplotlib.pyplot as plt

# Create sample dataset with 100 employees
departments = (
    ["Sales"] * 13 +
    ["IT"] * 20 +
    ["HR"] * 20 +
    ["Finance"] * 20 +
    ["Marketing"] * 27
)

data = {
    "employee_id": list(range(1, 101)),
    "department": departments,
}

df = pd.DataFrame(data)

# Frequency of Sales department
count_sales = (df["department"] == "Sales").sum()
print("Sales Department Count:", count_sales)

# Histogram of departments
plt.figure(figsize=(6, 6))
df["department"].value_counts().plot(kind="bar")
plt.title("Department Distribution")
plt.xlabel("Department")
plt.ylabel("Count")
plt.tight_layout()
plt.savefig("hist.png")
